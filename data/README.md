# Data Directory

COCO subsets generated by `scripts/prepare_coco_subset.py` are not committed because they include downloaded images and annotation copies.

Use the commands in the main `README.md` to recreate:

- `data/coco_subset_smoke/`
- `data/coco_subset_100/`
- `data/coco_subset/`

The `custom_demo/` folder is kept as a placeholder for self-collected classroom, campus, or lab images.
